# Randomizer

Randomizer generates random strings for Elixir projects.

## Installation

Add `randomizer` to your list of dependencies

```elixir
def deps do
  [
    {:randomizer, "~> 1.1.0"}
  ]
end
```

## Documentation

You can find the [documentation online](https://hexdocs.pm/randomizer).
